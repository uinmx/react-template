# react-template

这个是一个React模板，主要包含以下技术/功能：

### 环境

```nodemon
- node.js 18.17.0

- pnpm 8.15.4
```

### 技术栈

- React
- Typescript
- axios
- zustand
- react-router
- Ant Design UI
- hooks
- commitList
- eslint
- prettier
- vite

### use

```bash
pnpm install

pnpm dev
```

### 主要功能

**1.代码格式化**

> 打开编辑器保存时自动格式化代码选择（prettier）规范

**① 手动检查脚本:**

```bash
# eslint 规范检测
pnpm lint

# eslint 修复
pnpm lint:fix

# prettier 规范检测
pnpm format:check

# prettier 修复
pnpm format:code
```
